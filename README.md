# Laptop_Price_Prediction
Predicting the price of laptops by taking different features of laptops into consideration. A lot of preprocessing is done as there're a lot of unique values in each feature. 
## ML algorithms used :
-> Linear Regression

-> Ridge Regression

-> Lasso Regression

-> Decision Tree

-> KNN

-> RandomForest

-> Xgboost

-> Adaboost

-> Gradient boost

-> Catboost
